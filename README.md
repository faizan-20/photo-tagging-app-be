# Server setup
1. Clone this repository ```git clone https://github.com/faizan-20/photo-tagging-app-be```
2. Navigate to project folder ```cd photo-tagging-app-be```
3. Install the dependencies ```npm install```
4. Run the server ```npm run dev```
